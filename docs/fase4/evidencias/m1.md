# Evidência da M1

## 1. Procedimento

O inventário corresponde ao diretório `no_fluxo_frontend_svelte/src`, no commit `ba6db878b9dfa36fb034916612c4cf58ddf43475`.

Regras aplicadas:

1. componentes Svelte independentes foram contados por arquivo;
2. componentes internos de uma família em `components/ui/<familia>/` foram agrupados pelo módulo público `index.ts`;
3. hooks, utilitários, actions e estilos compartilhados foram contados por módulo;
4. arquivos agregadores, como `utils/index.ts`, não foram contados como ativo adicional;
5. um ativo foi considerado reutilizado quando importado por dois ou mais arquivos de contexto distintos;
6. imports internos de uma família de UI não foram considerados contexto externo.
