# NoFluxoUNB: Avaliação de Qualidade de Software

O **NoFluxoUNB** é uma plataforma desenvolvida para auxiliar estudantes da Universidade de Brasília no planejamento de suas trajetórias acadêmicas. Através de um fluxograma interativo e um chatbot inteligente, o sistema recomenda disciplinas e organiza o fluxo curricular de forma dinâmica. Dentro de sua interface é possível:

- Visualizar o fluxograma curricular de todos os cursos de graduação da UnB.
- Obter recomendações personalizadas de disciplinas com base no histórico acadêmico do estudante e interesse em áreas específicas.
- Compreender as relações de pré-requisitos e correquisitos entre as disciplinas, facilitando o planejamento do semestre.
- Comparar integralização curricular entre diferentes cursos, auxiliando na escolha de disciplinas optativas e possível transferência.

## Informações Adicionais

Para visualizar a documentação localmente, siga as instruções abaixo:

1. Execute o comando:

   ```bash
   pip install mkdocs
   ```

2. Clone o repositório:

   ```bash
   git clone https://github.com/FCTE-Qualidade-de-Software-1/2026-1_T02_CAROL_SHAW.git
   ```

3. Execute o comando para iniciar o servidor local:
   ```bash
   mkdocs serve
   ```
